Política de Privacidad

Aplicación: Calendario WoodTools · Última actualización: junio de 2026

Calendario WoodTools es una aplicación de escritorio que permite a su propietario (WoodTools) organizar tareas y programar la publicación de contenido en sus propias cuentas de redes sociales (TikTok, Instagram, Facebook, Threads y YouTube). Esta política explica qué datos maneja la aplicación y cómo.

1. Datos que maneja la aplicación

  • Tareas y contenido programado: títulos, descripciones, fechas, horarios y archivos (imágenes y videos) que el usuario crea. Se guardan localmente en la computadora del usuario.
  • Credenciales de las redes (tokens de acceso): se guardan cifradas en la computadora del usuario para poder publicar en su nombre. No se transmiten a ningún servidor nuestro.
  • Contenido a publicar: los videos/imágenes y textos se envían directamente a las APIs oficiales de cada red social (TikTok, Meta, Google) únicamente para publicarlos en las cuentas del propio usuario.

2. Cómo usamos los datos

Los datos se usan exclusivamente para la función de la aplicación: programar y publicar contenido en las cuentas del propio usuario. No vendemos, alquilamos ni compartimos los datos con terceros, ni los usamos con fines publicitarios o de seguimiento.

3. Almacenamiento

Toda la información (tareas, archivos y credenciales cifradas) se almacena localmente en el dispositivo del usuario. La aplicación no tiene servidores propios que recolecten o guarden datos de los usuarios.

4. Servicios de terceros

Para publicar contenido, la aplicación se conecta a las APIs oficiales de: TikTok, Meta (Instagram, Facebook, Threads) y Google (YouTube). El uso de esos servicios está regido por sus respectivas políticas de privacidad.

5. Eliminación de datos

El usuario puede eliminar todos los datos en cualquier momento desinstalando la aplicación o borrando los archivos locales de la app. Puede revocar el acceso de la aplicación a sus cuentas desde la configuración de seguridad de cada red social.
