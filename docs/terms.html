Términos de Servicio

Aplicación: Calendario WoodTools · Última actualización: junio de 2026

1. Aceptación

Al usar la aplicación Calendario WoodTools (la "Aplicación"), el usuario acepta estos Términos de Servicio. La Aplicación es una herramienta de escritorio para organizar tareas y programar la publicación de contenido en las cuentas de redes sociales del propio usuario.

2. Uso permitido

El usuario se compromete a usar la Aplicación únicamente para publicar contenido en cuentas que le pertenecen o que administra con autorización, y a cumplir con los términos y políticas de cada red social (TikTok, Meta, Google).

3. Responsabilidad del contenido

El usuario es el único responsable del contenido que crea, programa y publica a través de la Aplicación. La Aplicación solo actúa como herramienta para enviar ese contenido a las APIs oficiales de las redes sociales en nombre del usuario.

4. Credenciales y seguridad

El usuario es responsable de mantener seguras sus credenciales de acceso. La Aplicación guarda los tokens de forma cifrada localmente y no los comparte con terceros.

5. Disponibilidad

La Aplicación depende de las APIs de terceros (TikTok, Meta, Google). No garantizamos la disponibilidad continua de esos servicios ni nos responsabilizamos por cambios o interrupciones de las plataformas externas.

6. Limitación de responsabilidad

La Aplicación se proporciona "tal cual". En la medida que lo permita la ley, no nos hacemos responsables por daños derivados del uso o la imposibilidad de uso de la Aplicación.
